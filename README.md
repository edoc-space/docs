# Пример Контента E-Doc

В этом каталоге лежит пример контента для сайта на E-Doc.

Контент специально хранится обычными файлами:

- `site.json` настраивает название сайта, логотип, верхнее меню и футер;
- `pages/` содержит отдельные страницы сайта на Markdown и MDX;
- `docs/` содержит дерево документации, которое открывается в `/docs`;
- `static/` содержит публичные файлы: изображения, логотипы, JSON-данные и другие assets.

Этот каталог можно забрать как стартовый пример для своей документации. Скопируйте его в E-Doc установку, замените демонстрационные страницы на материалы своего проекта и сохраните ту же файловую модель.

Контент отделен от кода E-Doc. Его можно хранить в отдельном Git-репозитории и обновлять независимо от движка.

Типовой сценарий:

```bash
git clone <your-docs-repository> local/storage/edoc
rm -r local/storage/.git
cd local/storage/edoc && git init
```

После этого редактируйте Markdown или MDX файлы и обновляйте сайт.
