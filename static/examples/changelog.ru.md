# Changelog

## v1.2.0 - 2026-07-13

Этот выпуск улучшает структуру больших документаций: разделы `generated-index` теперь могут иметь rich-вступление, а проекты с несколькими независимыми ветками могут выбирать документацию по умолчанию для `/docs`.

### Добавлено

- Rich intro для `generated-index` через `index.md`, `index.mdx` или явный `content.source`.
- Автоматический generated list после intro без ручного списка ссылок.
- Поддержка `sidebar: true` для независимых верхнеуровневых веток документации.
- Поддержка `default_sidebar: true` для редиректа `/docs` в нужную ветку.
- Scope для sidebar и `prev` / `next` внутри активной ветки.
- Режимы контейнера для отдельных страниц сайта через `container`: `fluid`, `constrained`, `wide`, `narrow`.

### Изменено

- Markdown-ссылки внутри intro теперь резолвятся относительно каталога раздела.
- Поиск и table of contents учитывают rich intro generated-index страницы.
- Хлебные крошки отдельных страниц выравниваются по тому же контейнеру, что и контент страницы.
- Документация по `container` дополнена практическими сценариями и ширинами контейнеров.
- Инфраструктурный deploy по умолчанию обновлен на стабильный тег `v1.2.0`.

### Удалено

- Временный live-пример `rich-index` и пустая тестовая страница удалены из пользовательской навигации.

## v1.1.0 - 2026-07-10

Этот выпуск расширяет E-Doc от рендера документации до более полноценного рабочего пространства: в ядре появились preview изображений, video embeds и отдельная входная страница Help Center.

### Добавлено

- Полноэкранный preview для изображений в Markdown и MDX.
- Группировка изображений в галереи по `gallery` id, включая изображения в разных местах страницы.
- Ленивая загрузка видео для YouTube, RuTube и VK Video.
- MDX-компоненты `HelpCenter`, `HelpSection`, `HelpLink` и `HelpFaq`.
- Поиск через `searchProvider` с режимами `memory`, `static` и API-ready provider.
- Static endpoint поискового индекса для ленивого поиска на странице Help Center.

### Изменено

- Изображения в документации теперь выводятся с подписью, оформлением и ограничениями размера для горизонтальных и вертикальных кадров.
- В документацию по Markdown/MDX добавлены примеры медиа.
- В документацию по страницам сайта добавлен пример Help Center и будущий формат `api` search provider.

## v1.0.0 - 2026-07-01

Первый стабильный выпуск E-Doc закрывает базовый сценарий: запустить self-hosted сайт документации, хранить контент в Markdown/MDX и расширять страницы React-плагинами без изменения core-кода.

### Добавлено

- Публичный сайт документации с главной страницей, отдельными страницами и разделом `/docs`.
- Файловая модель контента: языковые папки, `pages`, `docs`, `static` и `site.json`.
- Поддержка Markdown и MDX для страниц и документации.
- Базовые MDX-блоки для главных страниц и контентных секций.
- Локальный слой плагинов без изменения core `package.json`.
- Первые плагины: OpenAPI reference, Changelog и DirectoryTree.
- Документация по установке, структуре контента, плагинам, локализации и self-hosted использованию.
